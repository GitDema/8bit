8Bit

8Bit is a Javascript module that transforms images into pixelated versions in canvas.

Usage Example

HTML

<canvas id="canvas"></canvas>

Javascript

var img = new Image();
img.onload = function () {
    eightBit(document.getElementById('canvas'), img, 10);
};
img.src = 'http://foobar.com/some_image.jpg';
